cd .. -> jdu o patro výš
cd začátek složkky + tab -> posune mě to do složky níže

dir -> ukáže mi to vše, co je v té dané složce
git --version -> sdělí mi to, jakou mám verzi gitu
git init -> slozka ma souviset s GITem

git add . -> přidej aktuální všechny změny na GIT
git add nazev slozky, kt chci commitnout -> přidej jen vybranou složku/soubor/změnu
git commit -m "První commit" -> přidáváme něco do historie -m -> message "vkladame popis, co delame"

ptá se to kdo jsem
git config --global user.name "Jmeno Prijmeni"
git config --global user.email "my email"

git status -> dá mi to aktuální info co je a není commit

nahravam novou slozku z compu
git remote add origin https://github.com/PajaCzechita/vizitka.git
git branch -M master
git push -u origin master
